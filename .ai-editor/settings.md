# AI Agent — 1Cfresh

## Инструкции для AI

### Правила общения
- ZERO FLUFF: Никаких приветствий, извинений, вступлений ("Конечно", "Надеюсь, поможет")
- MINIMAL REASONING: Не пиши ход мыслей — сразу к решению. Рассуждения оборачивай в <think>...</think>
- Текст — только если нужен для неочевидного пояснения
- Отвечай на языке вопроса (обычно русский)

### Правила работы с кодом
- FRUGALITY: Никогда не выводи весь файл, если изменилась лишь часть. Используй <replace> для точечных правок
- MODERN STACK: ES6+, актуальные версии библиотек, чистый безопасный код
- Без лишних абстракций, без обработки ошибок для невозможных сценариев
- Без комментариев в коде, если смысл не неочевиден

### Описание проекта
**1Cfresh** — платформа автоматизации выставления счетов с интеграцией 1С:Фреш и Контур.Диадок

**Backend (Python/FastAPI):**
- FastAPI + SQLAlchemy (async) + PostgreSQL/MySQL
- Alembic для миграций
- Celery + Redis для фоновых задач
- SMTP, Telegram polling, REST API для интеграций

**Frontend (Next.js/React):**
- Next.js 14.2.3, React 18 + TypeScript
- Radix UI + TailwindCSS, Zustand, TanStack React Table

**Основные модули:**
- Счета + расписания, Документы + DMS
- Портал клиентов, Чаты + Telegram
- Авторизация по тенантам

**Инфраструктура:**
- Docker Compose (PostgreSQL, Redis, API, Worker, Beat, Nginx)
- 94 deploy-скрипта для фич и фиксов

**Структура:** `/app` (backend), `/frontend` (Next.js), `/alembic` (миграции), `/nginx`, `/pdfs`
